## jogo-roleta
Um jogo simples em Python onde você tenta acertar números sorteados aleatoriamente para ganhar pontos!
## 📋 Descrição

O jogador participa de um jogo de roleta onde:
- Dois números entre 1 e 10 são sorteados aleatoriamente
- Se os números forem **iguais**, você ganha **5 pontos**
- Se forem diferentes, você não ganha pontos
- Você pode continuar jogando quantas vezes quiser
- Ao final, seu total de pontos é exibido

## 🚀 Como Usar

### Pré-requisitos
- Python 3.14

### Executar o Jogo

```bash
python jogo_roleta.py
